## 위로 검색
```
:?검색할 문자열
```

## 아래로 검색
```
:/검색할 문자열
```


## 다음 문자열 탐색
소문자 n


## 이전 문자열 탐색 (역방향 검색)
대문자 N


## 정규식 검색
```
/^정규식
```

(예시) test로 시작하는 문자열 찾을때
```
/^test
```
